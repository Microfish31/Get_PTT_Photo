# Get_PTT_Photo

## 說明
這是一個從 PTT 版上抓取圖片的工具。  
若頁數過多只抓前五頁(目前)
包含需要按下確認的頁面

## 設計
1. 獲取文章網址 (Thread)  需要頁數過多的判斷
2. 獲取圖片網址 (Thread)
3. 下載圖片 (Thread)

## 輸入
版名主題:    
ex: Stock  
關鍵字   
ex: 台積電
